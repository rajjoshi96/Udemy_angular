## Git Remote Basics

Open the folder in terminal 

    1. git init
    2. git remote add origin "complete path of github/web link.git"
    3. git pull origin main
    4. Make your changes
    5. git status
    6. git add .
    7. git commit -m "commit message"
    8. git push origin main
